# Troubleshooting

## Module Not Found

Run:

```bash
pip install -r requirements.txt
```

---

## Wrong Python Interpreter

Press:

```
Ctrl + Shift + P
```

Select:

```
Python: Select Interpreter
```

Choose:

```
venv/bin/python
```

---

## vector_store Missing

Run:

```bash
python scripts/ingest.py
```
