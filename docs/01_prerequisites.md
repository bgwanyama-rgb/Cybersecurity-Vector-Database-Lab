# Step 1 — Install Prerequisites

## Install Python

Check version:

```bash
python --version
```

Expected:

```
Python 3.10+
```

Download if needed:

https://python.org/downloads

---

## Install VS Code

Download:

https://code.visualstudio.com/

---

## Install VS Code Extensions

Open Extensions and install:

Required:

- Python (Microsoft)
- Pylance

Optional:

- GitHub Copilot
- Cursor
