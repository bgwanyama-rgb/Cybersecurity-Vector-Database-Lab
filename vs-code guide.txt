Here is a **clear, beginner-friendly, step-by-step guide to using your vector DB project with Visual Studio Code**, including environment setup, running the vector database, and using it with AI extensions like Cursor or Copilot.

---

# Step 1 — Install prerequisites 🧰

## 1. Install Python (if not installed)

Check:

```bash
python --version
```

You should see:

```bash
Python 3.10+
```

If not, install from:
[https://python.org/downloads](https://python.org/downloads)

---

## 2. Install VS Code

Download and install:
[https://code.visualstudio.com/](https://code.visualstudio.com/)

---

## 3. Install required VS Code extensions

Open VS Code → Extensions (left sidebar) → install:

**Required:**

* Python (Microsoft)
* Pylance (Microsoft)

**Recommended for AI integration:**

* Cursor (if using Cursor IDE)
* GitHub Copilot (optional)

---

# Step 2 — Extract and open the project 📁

## Extract the ZIP file you downloaded

Example location:

```bash
Downloads/vector-db-cybersecurity-lab-vscode-cursor
```

---

## Open folder in VS Code

In VS Code:

**File → Open Folder**

Select:

```bash
vector-db-cybersecurity-lab-vscode-cursor
```

You should see:

```bash
scripts/
data/
vector_store/
requirements.txt
README.md
```

---

# Step 3 — Open terminal in VS Code 💻

In VS Code:

**Menu → Terminal → New Terminal**

Terminal opens at bottom.

---

# Step 4 — Create Python virtual environment 🧪

Run:

```bash
python -m venv venv
```

This creates:

```bash
venv/
```

---

# Step 5 — Activate virtual environment ⚡

## Windows:

```bash
venv\Scripts\activate
```

## Mac/Linux:

```bash
source venv/bin/activate
```

You should now see:

```bash
(venv)
```

at start of terminal line.

---

# Step 6 — Install dependencies 📦

Run:

```bash
pip install -r requirements.txt
```

This installs:

* chromadb
* sentence-transformers

---

# Step 7 — Select Python interpreter in VS Code (IMPORTANT) 🧠

Press:

```bash
Ctrl + Shift + P
```

Type:

```bash
Python: Select Interpreter
```

Choose:

```bash
./venv/bin/python
```

or Windows:

```bash
venv\Scripts\python.exe
```

This ensures VS Code uses the correct environment.

---

## Problem: Module not found

Fix:

```bash
pip install -r requirements.txt
```

---

## Problem: Wrong Python interpreter

Fix:

```bash
Ctrl+Shift+P → Python: Select Interpreter
```

Fix:

```bash
python scripts/ingest.py
```
